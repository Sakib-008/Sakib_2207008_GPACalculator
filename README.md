# GPA Calculator - JavaFX Application

A desktop application built with JavaFX to help students calculate their Grade Point Average (GPA) by entering courses, credits, and grades. The app supports weighted GPA calculation, editing/deleting entered courses, and exporting results into a text file.

---

## Features

- Set **Required Credits** before adding courses.
- Add courses with the following details:
  - Course Name
  - Course Code
  - Credit Hours
  - Teachers
  - Grade
- Edit or delete previously added courses.
- Weighted GPA calculation: GPA = sum(credit Ã— grade points) / total credits.
- Reset all data to start fresh.
- Export course list and GPA to a text file.
